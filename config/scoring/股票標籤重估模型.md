核心概念是：

**標籤重估 \= 公司原本的市場分類，正在被新聞、營收、產品、法人資金重新歸類到更高估值的主題。**

例如：

光洋科  
 舊標籤：貴金屬／靶材／材料股  
 新標籤：AI 儲存、HAMR、半導體材料、關鍵金屬回收

中探針  
 舊標籤：探針／連接器／電子零組件  
 新標籤：AI 高功率測試、Burn-in Socket、MEMS 探針、先進封裝測試

---

# **一、可以轉成三層模型**

## **第一層：程式量化篩選**

先用數據找出「可能正在重估」的股票。

條件可以這樣設：

| 類別 | 條件 |
| ----- | ----- |
| 股價 | 站上 21MA，且 21MA 開始上彎 |
| 量能 | 突破日成交量 \> 20 日均量 1.5 倍 |
| 營收 | 近 3 個月營收 YoY 為正，至少 1 個月 \> 20% |
| 營收轉折 | 最近 3 個月平均 YoY \> 前 6 個月平均 YoY |
| 毛利率 | 最近一季毛利率 \> 前一季，或近兩季改善 |
| 營益率 | 虧損收斂、轉正，或近一季明顯改善 |
| 籌碼 | 法人近 5 日買超，融資沒有暴增 |
| 位階 | 股價剛脫離底部，距離 60MA 不要太誇張 |

這一層是找出「有資格被 AI 判讀」的股票。

---

# **二、第二層：AI 做標籤比對**

AI 要做的不是算數字，而是判斷：

**這家公司是否正在從舊標籤，切換到新標籤。**

可以讓 AI 讀：

1. 公司業務描述  
2. 最新新聞  
3. 法說會內容  
4. 月營收說明  
5. 年報產品分類  
6. 近期市場熱門題材詞庫

然後輸出：

| 欄位 | 說明 |
| ----- | ----- |
| 舊標籤 | 市場過去怎麼看這家公司 |
| 新標籤 | 現在是否出現更高估值題材 |
| 標籤轉換證據 | 新聞、營收、產品、客戶、法說 |
| 轉換強度 | 0～5 分 |
| 是否只是蹭題材 | 是 / 否 |
| 是否有營收支撐 | 是 / 否 |
| 是否進入主流資金敘事 | 是 / 否 |

---

# **三、第三層：標籤重估分數**

可以設一個 **Re-labeling Score 標籤重估分數，滿分 100 分**。

## **1\. 題材新鮮度 20 分**

判斷公司是否從舊產業切到熱門主線。

高分例子：

* 傳統材料 → 半導體材料  
* 連接器 → AI 伺服器高速連接  
* 測試治具 → AI/HPC 測試耗材  
* 散熱零件 → 液冷 / AI 伺服器散熱  
* 電源零件 → AI 伺服器高功率電源  
* 傳產設備 → 半導體設備零組件  
* 回收處理 → 關鍵金屬循環經濟

## **2\. 營收驗證 25 分**

不是只看新聞，要看數字。

可量化條件：

* 近 3 個月營收 YoY 皆為正  
* 近 3 個月平均 YoY \> 20%  
* 最近月營收創 12 個月新高  
* 累計營收年增 \> 15%  
* 營收成長高於過去 6 個月平均

## **3\. 毛利 / 營益率驗證 20 分**

判斷是不是產品組合真的改善。

高分條件：

* 毛利率連兩季上升  
* 營益率由負轉正  
* 營益率虧損收斂  
* EPS 由虧轉盈  
* 高毛利產品占比提升

## **4\. 新產品 / 新客戶證據 20 分**

這是標籤重估最重要的證據。

高分條件：

* 新產品已驗證通過  
* 已小量出貨  
* 已導入大客戶  
* 已進入量產  
* 法說會明確說占比會提升  
* 不是只有「規劃中」

## **5\. 資金確認 15 分**

市場是否開始承認這個新標籤。

高分條件：

* 股價突破 21MA 或平台  
* 成交量放大  
* 法人近 5 日買超  
* 融資沒有暴增  
* 新聞開始出現新標籤關鍵字

# **四、可以寫成程式邏輯**

大概像這樣：

def relabeling\_score(stock):  
   score \= 0

   \# 1\. 營收驗證  
   if stock.revenue\_yoy\_3m\_all\_positive:  
       score \+= 8  
   if stock.revenue\_yoy\_3m\_avg \> 20:  
       score \+= 8  
   if stock.latest\_revenue\_12m\_high:  
       score \+= 5  
   if stock.revenue\_yoy\_3m\_avg \> stock.revenue\_yoy\_6m\_avg:  
       score \+= 4

   \# 2\. 財報驗證  
   if stock.gross\_margin\_qoq\_up:  
       score \+= 6  
   if stock.gross\_margin\_2q\_up:  
       score \+= 6  
   if stock.operating\_margin\_turn\_positive:  
       score \+= 5  
   if stock.eps\_turn\_positive:  
       score \+= 3

   \# 3\. 技術籌碼  
   if stock.close \> stock.ma21 and stock.ma21\_slope\_up:  
       score \+= 5  
   if stock.volume \> stock.volume\_ma20 \* 1.5:  
       score \+= 4  
   if stock.institutional\_buy\_5d \> 0:  
       score \+= 4  
   if stock.margin\_balance\_growth \< 5:  
       score \+= 2

   return score

這是程式能算的部分。

# **五、AI 要負責判斷的部分**

程式很難知道「這家公司是不是正在換標籤」，所以要交給 AI。

AI Prompt 可以這樣設計：

請根據以下資料，判斷這家公司是否正在發生「市場標籤重估」。

請輸出：  
1\. 舊市場標籤  
2\. 新市場標籤  
3\. 標籤轉換是否成立，評分 0\~5  
4\. 是否有營收或財報驗證  
5\. 是否只是蹭題材  
6\. 轉型證據  
7\. 最重要的風險  
8\. 是否值得列入飆股觀察名單

判斷標準：  
\- 新標籤必須對應高估值產業，例如 AI、半導體、先進封裝、矽光子、液冷、電源、機器人、軍工、衛星、儲能。  
\- 不能只有新聞口號，必須有產品、客戶、營收、法說會或財報證據。  
\- 若只有公司說投入，但沒有出貨或營收，評分不得超過 2 分。  
\- 若已有小量出貨或客戶驗證，最高可給 4 分。  
\- 若新產品已明顯貢獻營收或毛利，最高可給 5 分。

# **六、最後整合成選股流程**

完整流程可以這樣：

每日收盤後：

1\. 抓全市場股票資料  
2\. 用程式篩出：  
  \- 站上 21MA  
  \- 放量  
  \- 營收轉強  
  \- 毛利率改善  
  \- 法人轉買

3\. 對候選股抓新聞、公司簡介、法說會、月營收說明

4\. 丟給 AI 判斷：  
  \- 舊標籤  
  \- 新標籤  
  \- 是否正在重估  
  \- 是否有財報驗證

5\. 產生總分：  
  \- 財報轉折分  
  \- 技術突破分  
  \- 籌碼分  
  \- 標籤重估分  
  \- 題材熱度分

6\. 排名前 20 名進入觀察清單

# **七、最實用的選股公式**

我會建議你最後做成這樣：

飆股候選分數 \=  
營收轉折 25%  
\+ 毛利/獲利改善 20%  
\+ 技術突破 20%  
\+ 籌碼乾淨度 15%  
\+ 標籤重估 20%

其中 **標籤重估分數** 是 AI 給的。

---

# **八、判斷標準一句話**

這套模型不是找「有 AI 題材的股票」，而是找：

**原本被市場低估成傳統產業，但近期營收、產品、新聞與技術面同時證明，它正在被重新歸類到高估值主線的股票。**

這件事可以做成程式，而且很值得做。

程式負責找「異常」，AI 負責判斷「異常背後是不是標籤重估」。
