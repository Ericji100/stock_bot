# 財務硬指標評分規則

本規則用於 AI 評估財務、營收、毛利、EPS、現金流與存貨等硬指標。這些分數必須以已提供資料與可靠來源為基礎，不得用題材故事或市場情緒補高。

## 核心面向

1. 營益率與營業利益品質。
2. 月營收 YoY、MoM 與近 3～6 個月趨勢。
3. EPS 與稅後淨利成長。
4. 毛利率、營益率是否改善或惡化。
5. 自由現金流與現金流穩定性。
6. 存貨週轉與庫存風險。

## 加分條件

- 月營收連續改善，且不是單月一次性因素。
- 毛利率或營益率明確改善。
- EPS 與本業獲利同步改善。
- 現金流與獲利品質一致。
- 存貨週轉改善，庫存風險降低。

## 扣分條件

- 月營收 YoY 轉弱或連續下滑。
- 毛利率、營益率或 EPS 惡化。
- 獲利主要來自業外或一次性項目。
- 存貨上升但營收未同步改善。
- 財報、月營收或現金流資料不足。

## 限制

- 財務硬指標資料不足時，不得給高財務分。
- 題材、新聞、社群或技術突破不得補高財務硬指標。
- 若財務與題材互相矛盾，必須明確說明矛盾並降低確定性。
