# 股票標籤重估模型規則

本規則用於判斷股票是否正在發生「舊市場標籤 → 新市場標籤」的價值重估。

## 核心問題

1. 舊市場標籤是什麼？
2. 新市場標籤是什麼？
3. 新標籤是否具備更高估值想像？
4. 標籤轉換是否有產品、客戶、營收、財報、公告、法說會或正式新聞支撐？
5. 是否只是同產業沾邊、蹭題材或短線情緒？

## 評分面向

1. 題材新鮮度。
2. 營收驗證。
3. 毛利 / 營益率驗證。
4. 新產品 / 新客戶證據。
5. 資金確認。
6. 反證與資料缺口。

## 高重估條件

- 舊標籤與新標籤轉換明確。
- 新產品、新客戶、新應用或新產業趨勢已有實質證據。
- 營收、毛利、EPS、法人籌碼或技術面至少部分驗證。
- 反證少，資料完整度高。

## 降分條件

- 沒有明確標籤轉換。
- 只有新聞、社群或短線情緒。
- 缺乏產品、營收、財報、公告或法人籌碼支撐。
- 存在公司澄清、產品無關、營收占比低、毛利惡化或需求不如預期。

## 蹭題材判斷

若新聞只提產業、不提公司；公司只是在同產業；題材缺少產品/客戶/營收支撐；或主要來自社群與短線資金，需標示為「蹭題材風險偏高」。
